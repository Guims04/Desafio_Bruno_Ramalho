DESAFIO

1 - Criar um método que recebe uma string e retorna somente as letras maiusculas

2 - Criar um método que receba um CPF e outro que receba um CNPJ e retorne se o número é válido

3 - Criar método que receba o CEP e retorne um objeto com o endereço

* Todos os métodos precisam estar bem documentados com comentários explicando cada linha de execução.

** Os métodos precisam estar em Javascript e se quiser também em outra(s) linguagem(ns).

* Se possível construir interface com React, Angular ou Vue.

** Opcionalmente também pode ser feito teste unitário para cada método desenvolvido.

*** Os códigos precisam estar publicados em repositório no github.com e enviado o link do mesmo.
